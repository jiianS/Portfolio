공공자전거 데이터를 이용한 분석 사이트

서울시 공공자전거 공공데이터를 이용한 분석 사이트

2018.08.08 - 2018.09.05

빅데이터 분석 및 시각화 처리를 위한 자바 스프링 개발자 양성과정 에서의 마지막 개인 프로젝트이다. 100개 이상의 데이터를 분석하여 분석한 결과를 시각화 하는 사이트를 만드는 것을 목표로 제작하게 되었다. 서울시 공공자전거의 공공데이터를 이용하여 대여소 위치를 지도에 나타내고 대여소 정보와 대여소 월별 이용건수를 분석하게 구현하였다. 또한, 월별 데이터를 이용하여 연령대 / 성별 로 이용건수와 이동거리를 분석하여 시각화하는 것을 구현하였다. 추후에 이용자 DB를 구축하여, 회원 관리 및 회원정보 확인 및 수정할 수 있도록 구현하였다.
아파치 서버를 구축하였으며 스프링 프레임워크와 하둡을 이용하였고 언어로는 자바, 마리아디비, html, jquery등을 사용하였다

발표왕

PPT Control Application for android

PPT Control Application for andriod

2014.03.24 - 2014.11.14
학교 졸업작품으로 [PPT Control] 어플리케이션 구현하였다. 당시 app시장에는 PPT control Application은 많았지만, ppt 화면을 받아오는 무료 Application 보기 어려웠기에 'PPT 화면이 스마트폰 화면에 보인다면?' 초점을 두고 프로젝트를 시작하였다.
개발툴은 Eclipse, Photoshop, Illustrator 이용, Language는 Android, JAVA를 이용하였다.
UDP통신을 이용하여 server.exe을 실행하면 따로 IP 주소를 찾지 않아도 IP주소를 이용할 수 있게 구현 2. 펜과 마우스 기능 3. VIEW _ PPT화면이 스마트폰에 보여지는 기능을 갖추었다.

5명으로 팀이 구성되었으며 팀장으로 프로젝트 총괄, UDP통신 코딩, 발표 자료 작성 및 보고서 작성을 맡았다.
